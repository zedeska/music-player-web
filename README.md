# Web music player

This is a high quality web music player with many quality of life features. It uses https://api.yams.tf API
## Features

- Multiple audio qualities - You can choose between 2 audio quality.
- Concurrent download - Downloads 3 (minimum) songs at the same time.
- Selective download - You can select specific track to download (i.e In an album).
- Custom file template - The downloaded track/album/playlist folder structure and file names can be customized.
- Music queue - Easy to use song queue. Can be re-arranged by drag and drop.
- Audio controls - Simple and efficient audio controls.
- File tagging - .mp3 and .flac file are tagged with the corresponding metadata.
- PWA - Progressive web app, can be installed on mobile.
- Responsive and simple design.
